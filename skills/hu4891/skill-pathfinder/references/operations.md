# 生态推荐与运营模块 (Ecosystem Operations)

本模块旨在保持系统长期的功能新鲜感，不断激发用户的主动探索欲。

## 1. 每日技能雷达 (Daily Radar)
- 设定定时或定频的概念，保持对网络上最新发布的高级技能（如：最具创意、最近突然爆发的大热门 Skill）的监测扫描和同步。

## 2. 智能打扰控制 (No Spam) 规则
- 绝对不可以像骚扰广告一样强行推送。必须设置极其严苛的打扰阈值。
- 如果当天雷达捕捉到的内容质量低下、重复造轮子，则宁愿保持绝对的静默，不发一言。
- 仅当发现了具有高度价值的新玩意的时，通过高度内敛的“每日精选”或“本周看点”的聚合卡片形式推送给用户。

## 3. 分类聚合推荐
为避免新发现的优秀扩展呈现散乱的状态，向用户推送时务必按照用户可理解的生活/工作场景进行打包和归类。
常见的场景标签分类包括但不限于：
- 🛠️ **效率提升**（涉及自动化文档处理、会议纪要等）
- 🤡 **趣味搞怪**（表情包生成器、夸夸机器人等）
- 💻 **专业硬核**（网络抓包、架构生成、金融回测等）

这样可以让用户像逛超市一样，根据自己当下的心情与急需的场景去按需挑选、探索和浏览。
