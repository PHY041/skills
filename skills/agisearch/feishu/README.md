# 飞书

> 这不是一个简单的飞书桥接工具，而是你的数字指挥中枢。

飞书技能专为中国企业高压协作环境设计，理解“分寸”与“效率”两套并行规则，把消息洪流、审批链、会议纪要、多维表格与日程安排，压缩成有优先级、可执行的行动链。

## 它能做什么

这个技能帮助你：

- 从群聊中提取真正需要你行动的消息
- 对审批进行预审、催办与卡点识别
- 从会议中自动提取决策与待办
- 让多维表格从静态记录变成动态决策层
- 自动追踪文档结论与更新变化
- 协调日程冲突并保护深度工作时间
- 根据层级和关系自动调整沟通分寸

## 它不是做什么

这个技能不是：

- 简单的消息转发器
- 泛化的办公自动化脚本
- 越权访问飞书数据的后台机器人
- 法务、财务、合规或人事裁决系统

## 适合谁

适合：

- 消息量巨大、群聊复杂的管理者
- 要处理审批、会议、周报、OKR 的中层角色
- 跨部门协作频繁、信息分散严重的团队
- 希望把飞书变成“决策中枢”而不是“消息入口”的组织

## 核心价值

飞书不是让你看更多信息。  
飞书应该让你更快做出正确动作。

这个技能的目标，不是“更智能地显示信息”，而是：

- 帮你少翻消息
- 少追人
- 少补纪要
- 少忘更新
- 少在错误的事情上消耗注意力

最终把飞书从协作噪音场，变成一个冷静、高效、有分寸的执行中枢。
