## 安装步骤

1. 在你的 OpenClaw `workspace/skills/` 新建文件夹 `hk-content-matrix`
2. 把本文件夹所有文件放进去
3. 重启 OpenClaw 或运行 `claw reload hk-content-matrix`
4. 给 Agent 权限：Browser Control + X/Twitter + 小红书/抖音网页版登录
5. 启动命令：`启动香港内容矩阵`
